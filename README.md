# Titanic Survival Prediction

## Objective
The objective of this project is to predict whether a passenger survived the Titanic disaster using Machine Learning algorithms based on passenger details such as age, gender, ticket class, fare, and family information.

---

## Technologies Used
- Python
- Jupyter Notebook
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

---

## Machine Learning Algorithms Used
This project uses classification algorithms such as:

- Logistic Regression
- Decision Tree Classifier
- Random Forest Classifier

---

## Features Used
The prediction is based on features like:

- Passenger Class (Pclass)
- Gender (Sex)
- Age
- Fare
- SibSp (Siblings/Spouses aboard)
- Parch (Parents/Children aboard)
- Embarked Location

---

## Workflow
1. Data Collection
2. Data Cleaning
3. Exploratory Data Analysis (EDA)
4. Feature Engineering
5. Data Visualization
6. Model Training
7. Model Evaluation
8. Survival Prediction
